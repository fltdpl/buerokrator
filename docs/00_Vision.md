# Vision

Buerokrator soll die manuelle Dokumentenablage weitgehend automatisieren.

## Ausgangssituation

Dokumente werden per Post empfangen und eingescannt.

## Zielprozess

1. Dokument wird eingescannt
2. Datei landet im Inbox-Ordner
3. OCR extrahiert den Text
4. Dokument wird analysiert
5. Datei wird korrekt benannt
6. Datei wird archiviert
7. Daten werden gespeichert
8. Steuerdaten werden bereitgestellt

## Erfolgskriterien

- Weniger manuelle Ablage
- Schnelles Wiederfinden von Dokumenten
- Vollständige Dokumenthistorie
- Steuerdaten auf Knopfdruck
