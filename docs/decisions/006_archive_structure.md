# Entscheidung  006
  
Dokumente werden nach Jahr und Kategorie archiviert.  
  
## Struktur  
  
archive/  
  
- Jahr  
	- Steuern  
	- Gesundheit  
	- Versicherungen  
	- Arbeit  
	- Bank  
	- Vorsorge  
	- Wohnen  
	- Sonstiges  
  
## Begründung  
  
Dokumente sollen dort abgelegt werden,  
wo sie später gesucht werden.
