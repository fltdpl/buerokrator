# Entscheidung 006

Dokumente werden nach Jahr und Kategorie archiviert.

## Struktur

Kategorienamen aus `config/settings.yaml` (`archive.category_mapping`):

archive/

- Jahr
	- Rechnungen
	- Steuern
	- Versicherungen
	- Vorsorge
	- Bank
	- Wohnen
	- Sonstiges

## Begründung

Dokumente sollen dort abgelegt werden,
wo sie später gesucht werden.
